# Crypto Learning AI

Crypto Learning AI is a simple AI agent that helps beginners learn cryptocurrency concepts through interactive quiz questions.

## Features
- Simple crypto quiz
- Beginner friendly
- Lightweight AI learning assistant

## Example Question
What does BTC stand for?

A. Bitcoin  
B. Binance Token

## Purpose
This project demonstrates how an AI assistant can help users learn crypto knowledge through simple interactive questions.

## OpenClaw Inspiration
This project is a lightweight AI learning assistant inspired by the OpenClaw AI agent concept.
It demonstrates how a simple AI agent can interact with users and teach basic crypto knowledge.

## Relation to Binance Ecosystem
The tool supports crypto education and aligns with the mission of making cryptocurrency knowledge accessible to everyone.

## How to Run

Clone the repository:

git clone https://github.com/nguyentridlk01-hub/crypto-learning-ai

Go to the project folder:

cd crypto-learning-ai

Run the AI agent:

python agent.py

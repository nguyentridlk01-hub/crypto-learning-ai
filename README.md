# Crypto Learning AI

Crypto Learning AI is a simple AI agent that helps beginners learn cryptocurrency concepts through interactive quiz questions.

## Features
- Simple crypto quiz
- Beginner friendly
- Lightweight AI learning assistant

## Example Question
What does BTC stand for?

A. Bitcoin  
B. Binance Token

## Purpose
This project demonstrates how an AI assistant can help users learn crypto knowledge through simple interactive questions.

## Relation to Binance Ecosystem
The tool supports crypto education and aligns with the mission of making cryptocurrency knowledge accessible to everyone.
